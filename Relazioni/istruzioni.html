Start the system

Nano

Considerare il progetto it.unibo2019.TemaFinale.sprint4
  1. Eseguire il comando gradle -b build_ctxButler.gradle distZip
  2. Recarsi nella cartella it.unibo2019.TemaFinale.sprint4/buld/distribution e copiare nel Raspberry il file it.unibo2019.TemaFinale.sprint4-1.0
  3. Decomprimere il file
  4. Copiare nella cartella it.unibo2019.TemaFinale.sprint4/bin i seguenti file:
    • basicRobotConfig.pl decommentare solo la riga robot( realnano, "" ).
    • butlerMindKB.pl
    • config.pl
    • configRoom.pl
    • mappa.bin e mappa.txt se già disponibili, altrimenti lanciare da PC MainCtxRoomExpl.kt
    • nearTable.pl
    • planningRouteKB.pl
    • resourceModelButler.pl
    • sprint4.pl modificare indirizzo ip del server MQTT e dei vari contesti
    • sysRules.pl
    • hardwareConfiguration.properties
    • SonarAlone.c e compilarlo g++ SonarAlone.c -l wiringPi -o SonarAlone
    • startBuzzer.sh e stopBuzzer.sh modificare le proprietà dei file chmod 777 startBuzzer.sh chmod 777 stopBuzzer.sh
  5. Sempre nella cartella bin creare la cartella configuration/nano e copiarci dentro il file iotRobot.properties
  6. Nella cartella bin eseguire il comando sudo bash it.unibo.eclipse.qak.sprint4
  7. Sul PC eseguire il Frontend:
    1. Modificare i file mqttUtils.js e coapClient.js inserendo gli ip del server MQTT e dei server CoAP.
    2. Nella cartella del frontend eseguire il comando npm start
    3. Nel browser recarsi alla pagina localhost:3000

Virtual

Considerare il progetto it.unibo2019.TemaFinale.sprint4 Avviare l'ambiente virtuale.
  1. Installare Node.js
  2. Aprire un ternimare in node\WEnv\server ed eseguire 
    npm install
  3. Nella cartella node\WEnv\WebGLScene eseguire 
    npm install
  4. Nella cartella node\WEnv\server\src eseguire 
    node main 8999
  5. Aprire un browser e recarsi alla pagina 
    localhost:8090
Avviare il sistema:
  1. Modificare il file basicRobotConfig.pl decommentando solo la riga robot(virtual, "8999")
  2. Modificare il file sprint4.pl inserendo gli indirizzi ip
  3. Avviare il sistema (su Bash sudo bash it.unibo.eclipse.qak.sprint4, su Windows eseguire it.unibo.eclipse.qak.sprint4.bat
  4. Sul PC eseguire il Frontend:
    1. Modificare i file mqttUtils.js e coapClient.js inserendo gli ip del server MQTT e dei server CoAP.
    2. Nella cartella del frontend eseguire il comando npm start
    3. Nel browser recarsi alla pagina localhost:3000
